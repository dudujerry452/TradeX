## 2026-04

### 实时聊天系统 (WebSocket)
- 基于 `channels` + `daphne` 实现 WebSocket 实时通讯
- 新增 `ChatMessage` 模型，支持关联商品/订单的上下文聊天
- REST API：对话列表、历史消息、标记已读、未读数统计、消息同步
- WebSocket 端点：`ws://<host>:<port>/ws/chat/`
- 开发环境采用双端口模式：Django HTTP (7001) + Daphne WebSocket (7002)

### 论坛功能
- 新增论坛模块：`ForumCategory`、`ForumTag`、`ForumPost`、`ForumComment`、`ForumPostLike`
- 支持帖子发布、分类筛选、标签关联、点赞、评论
- 提供分页列表与帖子详情接口

### 卖家中心
