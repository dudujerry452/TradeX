# TradeX v0.1.0 更新报告

## 1. 推荐系统

### 个性化推荐
- 基于用户标签偏好的推荐算法
- 已登录用户：根据收藏行为计算标签偏好分数
- 未登录用户：返回热门推荐作为降级策略
- 支持分页加载，偏移量机制

### 热门推荐
- 综合热度算法：`view_count * 0.2 + sales_count * 0.3 + favorite_count * 0.3 + avg_rating * 20 * 0.2`
- 按热度分数倒序排列
- 支持分页参数

### 相似商品
- 基于共同标签数量的相似度算法
- 计算商品间共享标签数量排序
- 用于商品详情页"猜你喜欢"

### 推荐数据收集
- 商品浏览记录自动上报
- 收藏行为更新用户标签偏好
- 标签权重动态计算

## 2. 分页机制

### 后端分页
- 接口统一支持 `limit` 和 `offset` 参数
- 推荐接口：数据库切片 `products[offset:offset + limit]`
- 搜索接口：QuerySet 切片 `products[offset:offset + limit]`

### 前端分页
- 首页推荐Tab支持无限滚动加载
- 滚动到底部自动触发加载更多
- 每页固定 10 条数据
- 显示加载指示器

### 分页状态管理
```dart
int _currentOffset = 0;      // 当前偏移量
final int _pageSize = 10;    // 每页数量
bool _isLoadingMore = false; // 加载中状态
bool _hasMoreData = true;    // 是否有更多数据
```

### 分页触发逻辑
- 滚动位置超过 `maxScrollExtent - 100` 时触发
- 避免重复请求：`!_isLoadingMore && _hasMoreData`
- 追加数据模式：`products.addAll(newItems)`

## 3. 刷新按钮

### 位置与样式
- 位于发现页顶部 Tab 栏右侧
- 金色刷新图标 `Color(0xFFCE965B)`
- 加载时显示圆形进度指示器

### 刷新逻辑
- 重置分页状态：`offset = 0`, `hasMoreData = true`
- 清空当前商品列表
- 重新加载当前 Tab 数据
- 支持下拉刷新手势

### 刷新范围
- 推荐 Tab：重新获取个性化/热门推荐
- 最新 Tab：重新获取商品列表
- 分类筛选：刷新后保持当前分类

## 4. 模糊搜索

### 搜索接口
- `GET /api/products/search/?q={关键词}&limit=10&offset=0`
- 可选 `token` 参数获取收藏状态
- 匹配字段：商品名称、描述、分类
- 仅返回 `APPROVED` 状态商品

### 搜索匹配逻辑
```python
Product.objects.filter(
    Q(product_name__icontains=q) |
    Q(description__icontains=q) |
    Q(category__icontains=q),
    product_status='APPROVED'
).distinct()
```

### 前端搜索UI
- 搜索框右侧"搜索"按钮
- 支持键盘回车触发搜索
- 搜索结果顶部显示关键词提示条
- 支持清除搜索返回正常列表

### 搜索分页
- 搜索结果支持无限滚动加载
- 与推荐列表共用分页状态管理
- 切换搜索/正常模式时重置分页

## 5. 商品展示

### 商品卡片
- 双列网格布局，宽高比 0.75
- 显示商品图片、名称、价格、分类
- 已收藏商品显示红心标记

### 分类筛选
- 横向滚动分类列表
- 支持"全部"及预设分类
- 分类切换触发刷新

### 空状态处理
- 加载中：圆形进度指示器
- 无数据：图标 + 文字提示
- 搜索无结果：提示"未找到相关商品"
